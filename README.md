# ЧИСТОВИК — сайт-визитка клининговой компании

Многостраничный сайт для клининговой компании: 5 страниц с общей навигацией,
светлая швейцарская сетка, острые углы, один рабочий синий цвет. Фирменная
фишка — интерактивный слайдер «до/после» на главной и в кейсах.

**Задача:** представить компанию целиком (не одну услугу) и вызвать доверие —
чек-лист вместо «на глаз чисто».
**Стек:** чистые HTML + CSS + JavaScript (модули), без сборки и зависимостей.
Шрифты — Unbounded / Golos Text / JetBrains Mono (Google Fonts), фото — Unsplash.

## Страницы
`index.html` Главная (hero со слайдером, процесс, услуги, отзывы) ·
`services.html` Услуги и цены (чек-лист на каждую услугу) ·
`about.html` О компании (цифры, ценности, команда) ·
`cases.html` Кейсы (слайдеры до/после) · `contacts.html` Контакты (форма).

Шапка и подвал рендерятся одинаково на каждой странице из `config.js` через
`main.js` — активный пункт меню подсвечивается по `data-page` в `<body>`.

## Запуск локально
Нужен любой статический сервер (из-за ES-модулей `file://` не подойдёт):
```bash
python3 -m http.server 4552
# открыть http://localhost:4552
```

## Правка контента
Весь контент — в `config.js`: услуги и чек-листы, процесс работы, команда,
кейсы (до/после), отзывы, контакты. Правишь один файл — обновляются все
5 страниц разом.

## Слайдер «до/после»
Реализован без библиотек: `clip-path` + `pointer events`, один компонент
переиспользуется на главной и в кейсах (`data-ba` в разметке). Перетаскивание
работает мышью и тачем.

## Деплой на GitHub Pages
```bash
git init && git add . && git commit -m "cleaning site"
gh repo create chistovik-cleaning --public --source=. --push
```
Затем **Settings → Pages → Branch: main /(root)** → живой адрес вида
`https://<username>.github.io/chistovik-cleaning/`.
